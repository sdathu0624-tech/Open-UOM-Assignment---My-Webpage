# Advanced IT Learning Portal

A static web app for learning IT skills — track your progress and complete quizzes for each course.

## Features
- 📚 Course carousel covering HTML & CSS, JavaScript, Python, Networking, and Database
- ✅ Progress tracking with "Mark Complete" buttons for each course
- 📝 Quiz section
- 📄 Materials section
- 📬 Contact section

## Tech Stack
- HTML5
- CSS3
- JavaScript (vanilla)
